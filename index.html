HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Try to use semantic HTML elements when possible. It also assists with those using screen readers
  • Keep things structured and consistent. An h1 element should not be nested within h3 and so on. It also makes the code easier to parse.
  • Alt + B opens the current VScode html file in the default browser.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * is the universal selector and can be used to select all elements on a page.
  • The CSS styles page must be linked (should be in head element) in order for it to apply to the page.
  • Do Not Repeat Yourself.

Git

  • git clone <repo SSH value>: clones the repo locally, creating a directory in the current folder.
  • code . : opens VScode with the explorer at the current directory.
  • git status: shows the current branch as well as any changes.
  • git add -A: stages the modifications to the current branch. -A adds all changes. There are also a multitude of other options with different effects.
  • git commit -m "add starter code": Commits staged changes to branch.
  • git pull origin main: Important for syncing local branch with GitHub. Pull brings in the modifications different from our local branch, origin points to the GitHub repo, and main indicates what branch we are pulling from.
  • git push origin <branch name>: Pushes changes to remote branch. From GitHub, this branch can now be pulled into the main branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function must be called in order for it to carry out its purpose.
  • Math.Random() can be used to generate a "random" number between 0 and 1. Math.floor() rounds down and returns a number that is less than or equal to the number given (so 4.8 would output 4)
